# Tableau_Study
---
다양한 도메인의 데이터를 바탕으로 여러 형태의 Tableau 대시보드를 제작한 학습 기록 저장소입니다.  

실제 실무에서 사용하는 형태의 지표 구성, KPI 카드, 필터/파라미터,  
차트 조합(Line, Bar, Area, Scatter, Heatmap 등)을 직접 구현하며  
**대시보드 설계 능력과 시각화 경험을 쌓는 것**을 목표로 했습니다.

## 🧰 What I Practiced
- 핵심 KPI·지표 테이블·히트맵 등 주요 지표 시각화 구성
- 데이터 구조에 맞는 다양한 차트(Line, Bar, Area, Scatter, Waterfall 등) 활용
- 필터·파라미터를 이용한 동적 분석 대시보드 설계
- 계산 필드 및 Table Calculation(YoY, 비율, 누계 등)을 통한 지표 재가공
- UX 중심 레이아웃 구성 및 시트 간 액션을 활용한 대시보드 상호작용 구현
