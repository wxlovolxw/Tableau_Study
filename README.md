# Tableau_Study
---
다양한 도메인의 데이터를 바탕으로 여러 형태의 Tableau 대시보드를 제작한 학습 기록 저장소입니다.  

실제 실무에서 사용하는 형태의 지표 구성, KPI 카드, 필터/파라미터,  
차트 조합(Line, Bar, Area, Scatter, Heatmap 등)을 직접 구현하며  
**대시보드 설계 능력과 시각화 경험을 쌓는 것**을 목표로 했습니다.

## 🧰 What I Practiced
- 여러 차트 유형 활용(Bar, Line, Area, Scatter, Heatmap, Pie 등)
- KPI 카드·지표 테이블·텍스트 히트맵 구현
- 필터/파라미터(연월, 지표 선택, 지역 선택 등)를 통한 동적 분석
- 계산 필드/Table Calculation을 이용한 지표 재가공
- 사용자 가독성을 고려한 레이아웃 구성 및 대시보드 UX 디자인
